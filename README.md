# Student Placement Management System

## Overview
The Student Placement Management System is a desktop application developed using Python, Tkinter, and SQLite. It helps manage student records, company details, placement drives, and selected students through a user-friendly interface.

## Technologies Used
- Python
- Tkinter
- SQLite

## Features
- Admin Login
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Manage Companies
- Placement Drives
- Reports
- Student Login
- Selected Students
- View Selected Students

## How to Run
1. Install Python.
2. Download the project.
3. Run `database.py` to create the database.
4. Run `login.py` to start the application.
